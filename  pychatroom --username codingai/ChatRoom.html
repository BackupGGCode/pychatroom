1 Overview

The program is implemented in Python programming language with Server-Client architecture. The default communication is group communication, which means the messages sent by every client is multicasted to every other clients. If one client want to chat with another client, it just needs to specify the client name in the message.

2 Features

  1. Server-Client Architecture;
  2. TCP Protocol
  3. Multi-Threaded server, each thread is responsible for a client connection;
  4. Multi-Threaded client, one for send message, another one for listening to server messages;
  5. Mutex is used on server to exclusively operate on client sockets;
  6. Special commands: "l" - list online users; "username>>>'message'" - send message to specific client;
  7. When one client connects to/disconnect from the server, all other clients are notified;
  8. GUI enabled;
  9. cross-platform.
